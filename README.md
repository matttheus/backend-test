# Run the project

```sh
docker-compose up
```

# Run tests

```sh
docker-compose run --rm web python -m unittest discover -s tests -v
```

# Requests Schema - POSTMAN

> The schema file is located at: `requests_schema.json`. You just need import it in the postman.
